> Traduzido pelo Google, abra um PR para ajudar a melhorar a tradução.

Implemente a função JavaScript `Array.includes` no sistema de tipos. Um tipo leva os dois argumentos. A saída deve ser um booleano `true` ou `false`.

Por exemplo:

```ts
type isPillarMen = Includes<['Kars', 'Esidisi', 'Wamuu', 'Santana'], 'Dio'> // expected to be `false`
```
