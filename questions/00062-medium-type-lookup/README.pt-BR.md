> Traduzido pelo Google, abra um PR para ajudar a melhorar a tradução.

Às vezes, você pode querer procurar um tipo em uma união pelos seus atributos.

Neste desafio, gostaríamos de obter o tipo correspondente pesquisando o campo `type` comum na união `Cat | Dog`. Em outras palavras, esperaremos obter `Dog` para `LookUp<Dog | Cat, 'dog'>` e `Cat` para `LookUp<Dog | Cat, 'cat'>` no exemplo a seguir.

```ts
interface Cat {
  type: 'cat'
  breeds: 'Abyssinian' | 'Shorthair' | 'Curl' | 'Bengal'
}

interface Dog {
  type: 'dog'
  breeds: 'Hound' | 'Brittany' | 'Bulldog' | 'Boxer'
  color: 'brown' | 'white' | 'black'
}

type MyDogType = LookUp<Cat | Dog, 'dog'> // expected to be `Dog`
```
