> Traduzido pelo Google, abra um PR para ajudar a melhorar a tradução.

Implemente um tipo `Sum<A, B>` que soma dois números inteiros não negativos e retorna a soma como uma string. Os números podem ser especificados como string, número ou bigint.

Por exemplo,

```ts
type T0 = Sum<2, 3> // '5'
type T1 = Sum<'13', '21'> // '34'
type T2 = Sum<'328', 7> // '335'
type T3 = Sum<1_000_000_000_000n, '123'> // '1000000000123'
```
