Implemente o tipo ```CheckRepeatedChars<S>``` que retornará se o tipo ```S``` contém caracteres duplicados?

Por exemplo:

```ts
type CheckRepeatedChars<'abc'>   // false
type CheckRepeatedChars<'aba'>   // true
```
