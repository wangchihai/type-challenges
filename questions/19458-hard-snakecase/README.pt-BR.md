> Traduzido pelo Google, abra um PR para ajudar a melhorar a tradução.

Crie um `SnakeCase<T>` genérico que transforma uma string formatada em **camelCase** em uma string formatada em **snake_case**.

Alguns exemplos:

```ts
type res1 = SnakeCase<"hello">; // => "hello"
type res2 = SnakeCase<"userName">; // => "user_name"
type res3 = SnakeCase<"getElementById">; // => "get_element_by_id"
```
