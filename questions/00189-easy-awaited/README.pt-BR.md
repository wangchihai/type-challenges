Se tivermos um tipo empacotado como Promise, como podemos obter o tipo que está dentro do tipo empacotado?

Por exemplo: se tivermos `Promise<ExampleType>` como obter ExemploType?

```ts
type ExampleType = Promise<string>

type Result = MyAwaited<ExampleType> // string
```
