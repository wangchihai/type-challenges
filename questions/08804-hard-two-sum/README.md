Given an array of integers `nums` and an integer `target`, return true if two numbers such that they add up to `target`.

For example

```ts
type sum1 = TwoSum<[3, 2, 4], 6> // true
type sum2 = TwoSum<[2, 7, 11, 15], 15> // false
```
