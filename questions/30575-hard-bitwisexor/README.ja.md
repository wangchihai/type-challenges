> この課題はGoogleが翻訳しました。翻訳品質改善のためのPRを募集しています。

2 つのバイナリ文字列リテラル型を受け取り、S1 と S2 のビットごとの XOR を表すバイナリ文字列を返す ```BitwiseXOR<S1,S2>``` を実装します。

例えば：

```typescript
BitwiseXOR<'0','1'> // expect '1'
BitwiseXOR<'1','1'> // expect '0'
BitwiseXOR<'10','1'>  // expect '11'
```
