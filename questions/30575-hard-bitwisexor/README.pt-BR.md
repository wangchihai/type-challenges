> Traduzido pelo Google, abra um PR para ajudar a melhorar a tradução.

Implemente ```BitwiseXOR<S1,S2>``` que usa dois tipos literais de string binária e retorna uma string binária que representa o XOR bit a bit de S1 e S2

Por exemplo:

```typescript
BitwiseXOR<'0','1'> // expect '1'
BitwiseXOR<'1','1'> // expect '0'
BitwiseXOR<'10','1'>  // expect '11'
```
