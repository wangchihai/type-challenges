型レベルの整数比較器を実装します。次のように、比較結果を示す列挙型を提供しました。

- `a` が `b` より大きい場合、タイプは `Comparison.Greater` である必要があります。
- `a` と `b` が等しい場合、タイプは `Comparison.Equal` である必要があります。
- `a` が `b` より小さい場合、タイプは `Comparison.Lower` である必要があります。

**`a` と `b` は、正の整数、負の整数、またはゼロにすることができ、一方が正でもう一方が負である場合もあることに注意してください。**
