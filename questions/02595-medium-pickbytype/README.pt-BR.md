> Traduzido pelo Google, abra um PR para ajudar a melhorar a tradução.

Em `T`, escolha um conjunto de propriedades cujo tipo pode ser atribuído a `U`.

Por exemplo

```typescript
type OnlyBoolean = PickByType<{
  name: string
  count: number
  isReadonly: boolean
  isEnable: boolean
}, boolean> // { isReadonly: boolean; isEnable: boolean; }
```
