> Traduzido pelo Google, abra um PR para ajudar a melhorar a tradução.

Existe uma função na linguagem C: `printf`. Esta função nos permite imprimir algo com formatação. Assim:

```c
printf("The result is %d.", 42);
```

Este desafio exige que você analise a string de entrada e extraia os espaços reservados de formato como `%d` e `%f`. Por exemplo, se a string de entrada for `"The result is %d."`, o resultado analisado será uma tupla `['dec']`.

Aqui está o mapeamento:

```typescript
type ControlsMap = {
  c: 'char',
  s: 'string',
  d: 'dec',
  o: 'oct',
  h: 'hex',
  f: 'float',
  p: 'pointer',
}
```
