> Traduzido pelo Google, abra um PR para ajudar a melhorar a tradução.

Implemente a versão do tipo ```Math.trunc```, que pega string ou número e retorna a parte inteira de um número removendo quaisquer dígitos fracionários.

Por exemplo:

```typescript
type A = Trunc<12.34> // 12
```
