> Traduzido pelo Google, abra um PR para ajudar a melhorar a tradução.

Implemente `TrimRight<T>` que pega um tipo de string exato e retorna uma nova string com o final do espaço em branco removido.

Por exemplo:

```ts
type Trimed = TrimRight<'   Hello World    '> // expected to be '   Hello World'
```
