> Traduzido pelo Google, abra um PR para ajudar a melhorar a tradução.

Implemente a função `any` preferida do Python no sistema de tipos. Um tipo pega o Array e retorna `true` se algum elemento do Array for verdadeiro. Se o Array estiver vazio, retorne `false`.

Por exemplo:

```ts
type Sample1 = AnyOf<[1, "", false, [], {}]> // expected to be true.
type Sample2 = AnyOf<[0, "", false, [], {}]> // expected to be false.
```
