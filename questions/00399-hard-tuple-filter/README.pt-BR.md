> Traduzido pelo Google, abra um PR para ajudar a melhorar a tradução.

Implemente um tipo `FilterOut<T, F>` que filtre itens do tipo `F` fornecido da tupla `T`.

Por exemplo,
```ts
type Filtered = FilterOut<[1, 2, null, 3], null> // [1, 2, 3]
```
