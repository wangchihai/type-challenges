> Traduzido pelo Google, abra um PR para ajudar a melhorar a tradução.

O conhecido método `split()` divide uma string em um array de substrings procurando um separador e retorna o novo array. O objetivo deste desafio é dividir uma string, usando um separador, mas no sistema de tipos!

Por exemplo:

```ts
type result = Split<'Hi! How are you?', ' '>  // should be ['Hi!', 'How', 'are', 'you?']
```
