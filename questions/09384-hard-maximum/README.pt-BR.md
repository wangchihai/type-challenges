### Descrição

Implemente o tipo `Maximum`, que recebe um tipo de entrada `T` e retorna o valor máximo em `T`.

Se `T` for um array vazio, ele retornará `never`. **Números negativos** não são considerados.

Por exemplo:

```ts
Maximum<[]> // never
Maximum<[0, 2, 1]> // 2
Maximum<[1, 20, 200, 150]> // 200
```
