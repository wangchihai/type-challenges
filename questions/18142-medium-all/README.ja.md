> この課題はGoogleが翻訳しました。翻訳品質改善のためのPRを募集しています。

リストのすべての要素が渡された 2 番目のパラメーターと等しい場合は true を返し、不一致がある場合は false を返します。

例えば

```ts
type Test1 = [1, 1, 1]
type Test2 = [1, 1, 2]

type Todo = All<Test1, 1> // should be same as true
type Todo2 = All<Test2, 1> // should be same as false
```
