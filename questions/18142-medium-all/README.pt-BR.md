> Traduzido pelo Google, abra um PR para ajudar a melhorar a tradução.

Retorna verdadeiro se todos os elementos da lista forem iguais ao segundo parâmetro passado, falso se houver alguma incompatibilidade.

Por exemplo

```ts
type Test1 = [1, 1, 1]
type Test2 = [1, 1, 2]

type Todo = All<Test1, 1> // should be same as true
type Todo2 = All<Test2, 1> // should be same as false
```
