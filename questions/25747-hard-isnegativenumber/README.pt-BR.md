> Traduzido pelo Google, abra um PR para ajudar a melhorar a tradução.

Às vezes, ao trabalhar com literais numéricos, precisamos descartar (ou impor) que o número fornecido seja um número inteiro positivo.

Para fazer isso, primeiro precisamos de saber se o número é negativo.

Escreva uma função de nível de tipo `IsNegativeNumber` que aceite um número `N` e retorne:

- `true` se `N` for negativo
- `false` se `N` for positivo
- `false` se `N` for `0`,
- `never` se `N` for `number`
- `never` se `N` for uma união
