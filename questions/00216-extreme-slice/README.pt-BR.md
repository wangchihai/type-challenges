> Traduzido pelo Google, abra um PR para ajudar a melhorar a tradução.

Implemente a função JavaScript `Array.slice` no sistema de tipos. `Slice<Arr, Start, End>` leva os três argumentos. A saída deve ser uma submatriz de `Arr` do índice `Start` a `End`. Índices com números negativos devem ser contados inversamente.

Por exemplo

```ts
type Arr = [1, 2, 3, 4, 5]
type Result = Slice<Arr, 2, 4> // expected to be [3, 4]
```
