Block、Element、Modifier 手法 (BEM) は、CSS のクラスの一般的な命名規則です。

たとえば、ブロック コンポーネントは `btn` として表され、ブロックに依存する要素は `btn__price` として表され、ブロックのスタイルを変更する修飾子は `btn--big` または `btn__price--warning` として表されます。

これら 3 つのパラメータから文字列結合を生成する `BEM<B, E, M>` を実装します。ここで、`B` は文字列リテラル、`E` と `M` は文字列配列 (空でも可) です。
