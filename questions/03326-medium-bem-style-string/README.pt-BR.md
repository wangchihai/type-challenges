> Traduzido pelo Google, abra um PR para ajudar a melhorar a tradução.

A metodologia Block, Element, Modifier (BEM) é uma convenção de nomenclatura popular para classes em CSS.

Por exemplo, o componente do bloco seria representado como `btn`, o elemento que depende do bloco seria representado como `btn__price`, o modificador que altera o estilo do bloco seria representado como `btn--big` ou `btn__price--warning`.

Implemente `BEM<B, E, M>` que gera união de strings a partir desses três parâmetros. Onde `B` é uma string literal, `E` e `M` são matrizes de strings (podem estar vazias).
