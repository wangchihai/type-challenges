实现一个类型 `LengthOfString<S>` 来计算模板字符串的长度（如 [298 - 字符串长度](https://tsch.js.org/298) 中所示）：

```ts
type T0 = LengthOfString<"foo"> // 3
```

该类型必须支持几百个字符长的字符串（通常的递归计算字符串长度受到TS中递归函数调用深度的限制，即最多支持45个字符左右的字符串长）。
