> Traduzido pelo Google, abra um PR para ajudar a melhorar a tradução.

O problema FizzBuzz é um teste clássico aplicado em entrevistas de codificação. A tarefa é simples:

Imprima inteiros de 1 a N, exceto:

- Imprima "Fizz" se um número inteiro for divisível por 3;
- Imprima "Buzz" se um número inteiro for divisível por 5;
- Imprima "FizzBuzz" se um número inteiro for divisível por 3 e 5.

Por exemplo, para N = 20, a saída deve ser:
`1, 2, Fizz, 4, Buzz, Fizz, 7, 8, Fizz, Buzz, 11, Fizz, 13, 14, FizzBuzz, 16, 17, Fizz, 19, Buzz`

No desafio abaixo, desejaremos gerar isso como um array de strings literais.

Para valores grandes de N, você precisará garantir que todos os tipos gerados o façam de forma eficiente (por exemplo, por
corretamente usando a otimização de chamada final para recursão).
