> Traduzido pelo Google, abra um PR para ajudar a melhorar a tradução.

Implemente a função JavaScript `Array.concat` no sistema de tipos. Um tipo leva os dois argumentos. A saída deve ser uma nova matriz que inclui entradas na ordem ltr

Por exemplo:

```ts
type Result = Concat<[1], [2]> // expected to be [1, 2]
```
