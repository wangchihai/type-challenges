Implemente um tipo ```IsTuple```, que recebe um tipo de entrada ```T``` e retorna se ```T``` é do tipo tupla.

Por exemplo:

```typescript
type case1 = IsTuple<[number]> // true
type case2 = IsTuple<readonly [number]> // true
type case3 = IsTuple<number[]> // false
```
