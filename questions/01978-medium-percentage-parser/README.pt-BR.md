> Traduzido pelo Google, abra um PR para ajudar a melhorar a tradução.

Implemente PercentageParser<T estende string>.
De acordo com a regularidade `/^(\+|\-)?(\d*)?(\%)?$/` para combinar T e conseguir três partidas.

A estrutura deve ser: [`plus or minus`, `number`, `unit`]
Se não for capturado, o padrão será uma string vazia.

Por exemplo:

```ts
type PString1 = ""
type PString2 = "+85%"
type PString3 = "-85%"
type PString4 = "85%"
type PString5 = "85"

type R1 = PercentageParser<PString1> // expected ['', '', '']
type R2 = PercentageParser<PString2> // expected ["+", "85", "%"]
type R3 = PercentageParser<PString3> // expected ["-", "85", "%"]
type R4 = PercentageParser<PString4> // expected ["", "85", "%"]
type R5 = PercentageParser<PString5> // expected ["", "85", ""]
```
