> Traduzido pelo Google, abra um PR para ajudar a melhorar a tradução.

Implemente o tipo `Absolute`. Um tipo que aceita string, número ou bigint. A saída deve ser uma sequência numérica positiva

Por exemplo

```ts
type Test = -100
type Result = Absolute<Test> // expected to be "100"
```
