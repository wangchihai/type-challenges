> Traduzido pelo Google, abra um PR para ajudar a melhorar a tradução.

Implemente um tipo que adicione um novo campo à interface. O tipo leva os três argumentos. A saída deve ser um objeto com o novo campo.

Por exemplo

```ts
type Test = { id: '1' }
type Result = AppendToObject<Test, 'value', 4> // expected to be { id: '1', value: 4 }
```
