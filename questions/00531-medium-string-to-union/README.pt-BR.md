> Traduzido pelo Google, abra um PR para ajudar a melhorar a tradução.

Implemente o tipo String to Union. Digite o argumento take string. A saída deve ser uma união das letras de entrada

Por exemplo

```ts
type Test = "123"
type Result = StringToUnion<Test> // expected to be "1" | "2" | "3"
```
