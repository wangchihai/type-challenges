Implemente um `RequiredByKeys<T,  K>` genérico que recebe dois argumentos de tipo `T` e `K`.

`K` especifica o conjunto de propriedades de `T` que deve ser definido como obrigatório. Quando `K` não é fornecido, todas as propriedades devem ser exigidas exatamente como o `Required<T>` normal.

Por exemplo

```typescript
interface User {
  name?: string
  age?: number
  address?: string
}

type UserRequiredName = RequiredByKeys<User, 'name'> // { name: string; age?: number; address?: string }

```
